# NIDAR Drone System Training

This repository contains my work for the **NIDAR (National Innovation Challenge for Drone Application and Research) Disaster Management Drone System project training at RISC**.

The project is divided into three main components:

* ROS2 software development
* CAD design of drone components
* Electronics design and integration

---

# Repository Structure

## ROS2 Packages

Location:

```
src/
```

Contains the ROS2 code for the drone system.

### drone1_pkg

Nodes implemented:

* simple_node
* altitude_publisher
* altitude_subscriber
* arm_server
* arm_client

### drone_sim

Contains ROS2 launch files to run multiple nodes.
